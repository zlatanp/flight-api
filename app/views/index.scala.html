Welcome to the Flight Advisor!

Endpoints

GET

/home returns home page.

POST

/user/login login new user to the system.

curl -X POST -H 'Content-Type: application/json' -d '{"name": "pera", "password": "pera1234"}' localhost:9000/user/login

POST

/user/logout logout logged user from the system.

curl -X POST -H 'Content-Type: application/json' localhost:9000/user/logout

GET

/city/all return list of all cities from the system. First request is example for getting list of all cities with all comments, and second request is example for getting list of all cities with specified number of comments per city.

curl -X GET -d '{}' localhost:9000/city/all
curl -X GET -d '{"comments" : 1}' localhost:9000/city/all

GET

/city/one return one city from the system. First request is example for getting all comments for specific city, and second request is example for getting specified number of comments for specific city.

curl -X GET -d '{"cityName" : "Belgrade"}' localhost:9000/city/one
curl -X GET -d '{"cityName" : "Belgrade", "comments" : 1}' localhost:9000/city/one

POST

/city/add add new city to the system.

curl -X POST -H 'Content-Type: application/json' -d '{"name": "Lyion", "country": "France", "description": "0,5 milion people"}' localhost:9000/city/add

POST

/city/comment add new comment for city. If comment already exists from logged in user, comment will be updated.

curl -X POST -H 'Content-Type: application/json' -d '{"cityName": "Belgrade", "comment": "Nice city!"}' localhost:9000/city/comment

DELETE

/city/deleteComment delete comment for city.

curl -X DELETE -H 'Content-Type: application/json' -d '{"cityName" : "Belgrade"}' localhost:9000/city/deleteComment

POST

/import import all airport and route information for existing cities to the system. (Only admin can access to this endpoint)

curl -X POST -H 'Content-Type: application/json' -d '{}' localhost:9000/import

GET

/flights get best flight between two cities

curl -X GET -H 'Content-Type: application/json' -d '{"from": "Belgrade", "to": "Dubai"}' localhost:9000/flights }
